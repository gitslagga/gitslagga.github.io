Atom的插件必备

发表于 2018-02-02 更新于 2025-06-06 分类于 DevTools

ATOM常用插件推荐

  1. simplified-chinese-menu ATOM的汉化插件
  2. file-icons 文件图标美化

  1. atom-beautify 代码格式一键美化

  2. last-cursor-position 光标自由切换到上一次/下一次编辑位置

  3. minimap 代码小地图

  4. sync-settings 插件备份、按键绑定备份

  5. activate-power-mode 输入时有震撼效果

  6. tortoise-svn svn插件

  7. atom-terminal-panel atom内置命令行工具

  8. Git /github

    git-plus—在 Atom 里面执行 Git 命令，不用来回切换终端和编辑器
    git-control—git面板 
    tree-view-git-status—文件夹git状态
    gist-it—快速分享代码到gist.github.com
    git-log-图形化git提交记录

  9. markdown-preview-plus—markdown预览

  10. vim-mode 支持vim模式

  11. script 在atom下运行脚本，支持多种开发语言。

  12. remote-edit 编辑机远程文件

  13. browser-plus 编辑器内置浏览器

  14. ask-stack 在atom快速提问stackoverflow

  15. regex-railroad-diagram 正则表达式图形化

  16. advanced-open-file 快速的打开文件或新建文件，同时支持路径补全

  17. quick-highlight 代码高亮

  18. platformio-ide-terminal 终端工具

  19. open-in-browsers 在安装过的任意浏览器打开代码

  20. merge-conflicts 在 Atom 里面处理合并产生冲突的文件

  21. linter 代码校验工具

  22. autocomplete-paths 填写路径的时候有提示

  23. linter 代码校验工具(必备)

前端插件推荐

  1. emmet 加快web开发速度，提供snippet(代码片段)、abbreviation expand(简写展开)功能。

  2. color-picker 在编辑器里面挑选颜色

  3. language-JavaScript-jsx 支持JSX语法

  4. atom-html-preview 实时预览HTML页面

  5. React

    react—语法支持 
    atom-react-native-autocomplete—React Native代码补全 
    nuclide—facebook基于atom开发的进行react开发的IDE

  6. language-babel 支持JSX语法

  7. atom-minify css、js代码压缩

  8. atom-ternjs js,nodejs,es6补全，高度定制化

  9. jshint 验证js和jsx语法是否正确

  10. csslint 验证css语法是否正确

  11. autoprefixer 自动为 CSS 属性添加特定的前缀

  12. autoclose-html 闭合html标签

  13. pigments 颜色显示插件

  14. less-autocompile less文件编译为css文件

php插件推荐

  1. php-cs-fixer php语法检查

  2. php跳转插件(类似其他编辑器的ctrl+click) 需要同时安装以下两个插件： 

    goto 
    goto-definition 鼠标光标放置在要跳转的类下，按下快捷键 ctrl-alt-enter

  3. php-debug php debug

  4. autocomplete-php php代码自动补全

python插件推荐

  1. python-tools

    Python 跳转插件

  2. python-debugger python debug

  3. autocomplete-python python代码自动补全

  4. linter-pep8 python编码规范检查
